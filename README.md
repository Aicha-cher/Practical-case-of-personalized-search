# Practical-case-of-personalized-search

## Introduction

Personalized search refers to web search experiences that are tailored specifically to
an individual’s interests by incorporating information about the individual beyond
the specific query provided. There are two general approaches to personalizing
search results, involving modifying the user’s query and re-ranking search results[1]
In this work, we will focus on the using a predefined users profiles that will be used
to expand users' queries in an effort to maximize the capture the user information
need
